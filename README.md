# sql-select-fundamentals
--Usar SELECT * puede afectar al rendimiento ya que trae todas las columnas de la tabla, incluso las que no se necesitan, y  esto aumenta el consumo de memoria.
--Tambien puede afectar a la seguridad, ya que puede exponer datos sensibles.
--Los alias son importantes porque permiten mostrar los nombres de las columnas de forma clara y comprensible para personas que no tienen conocimientos técnicos.
--Ejemplo: que la descripcion de la columna este en ingles, y no quede clara para todos los usuarios.
